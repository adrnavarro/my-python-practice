# Modules

This directory contains reusable Python modules and packages.

## Guidelines

- Create well-documented, reusable code
- Each module should have a clear, single responsibility
- Include docstrings for all public functions and classes
- Consider creating `__init__.py` files for packages

## Example Structure

```
modules/
├── __init__.py
├── my_module.py
└── my_package/
    ├── __init__.py
    └── submodule.py
```
